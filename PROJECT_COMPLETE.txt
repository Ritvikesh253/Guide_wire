API ENDPOINTS
=============

Auth & Registration:
  POST   /api/auth/otp/send              Send OTP
  POST   /api/auth/otp/verify            Verify OTP
  POST   /api/auth/login                 Login
  POST   /register/step1-4               4-step registration

Feature 1 - Zones:
  GET    /api/zones                      List all zones
  GET    /api/zones/{zoneId}             Get zone details

Feature 2 - Premium:
  POST   /api/premium/calculate          Calculate ML premium

Feature 3 - Claims & Disrupt:
  POST   /api/claims/manual              Create manual claim
  GET    /api/claims/trigger-disruption  Simulate disruption event

Feature 4 - Payouts:
  POST   /api/payouts/initiate           Start UPI payout
  GET    /api/payouts/{payoutId}/status  Check payout status

Feature 5 - Aadhaar:
  POST   /api/aadhaar/otp                Request OTP
  POST   /api/aadhaar/verify             Get e-KYC data
  POST   /api/aadhaar/verify-face        Verify face liveness

QUICKSTART
==========

Option 1: Local Execution (Windows PowerShell)
  .\startup.ps1                   # Starts all services
  .\startup.ps1 backend           # Just backend
  .\startup.ps1 ml                # Just ML service

Option 2: Docker Compose
  docker-compose up -d            # Start all (requires Docker)
  docker-compose down             # Stop all

Option 3: Manual Start
  Backend:  cd backend && java -jar target/payassure-backend-1.0.0.jar
  ML:       cd backend/ml-service && python app.py
  Mock API: cd backend/mock-disruption-api && python app.py
  Frontend: cd frontend && npm start
